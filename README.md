Este repositório congrega **exercícios, exemplos e singelos projetos desenvolvidos em C#**, concebidos com o propósito de **cultivar, aprimorar e consolidar os fundamentos da linguagem**.  
Serve, pois, como um espaço de estudo contínuo, reflexão técnica e progresso gradual.

---

## 📌 Sobre o Repositório

Aqui se encontram códigos voltados ao **estudo diligente da linguagem C#**, abordando, entre outros, os seguintes conceitos fundamentais:

- Declaração e uso de variáveis e tipos de dados  
- Estruturas condicionais (`if / else`)  
- Estruturas de repetição (`for`, `while` e `do while`)  
- Métodos e funções  
- Classes e objetos  
- Conceitos introdutórios de Programação Orientada a Objetos (POO)  
- Operações de entrada e saída de dados via console  

Os exemplos aqui reunidos são, em sua maioria, **simples e didáticos**, muitos deles acompanhados de comentários explicativos, cuidadosamente elaborados para favorecer a compreensão daqueles que se iniciam na linguagem.

---

## 🧠 Objetivo

Este repositório tem por finalidade:

- Consolidar o aprendizado da linguagem C#  
- Servir como fonte de consulta futura  
- Organizar exercícios realizados em ambiente acadêmico e extraacadêmico  
- Registrar, de forma progressiva, a evolução do conhecimento adquirido ao longo do tempo

## 🚀 Como Executar os Códigos

1. Clone o repositório:
```bash
   git clone https://github.com/Yunik-D/Csharp.git
```
2. Abra o projeto no Visual Studio ou Visual Studio Code
3. Certifique-se de possuir o .NET SDK devidamente instalado
👉 https://dotnet.microsoft.com/download
4. Execute o arquivo desejado por meio do console ou da IDE de sua preferência

## ✨ Contribuições

Embora este repositório possua fins primordialmente educacionais, toda contribuição construtiva é cordialmente bem-vinda.

São particularmente apreciadas iniciativas que visem:

- Aperfeiçoar a organização dos arquivos
- Acrescentar comentários explicativos ao código
- Introduzir novos exemplos ou exercícios didáticos

Para contribuir:

- Realize um fork do projeto
- Crie uma branch destinada à sua modificação
- Submeta um Pull Request descrevendo, com clareza, as alterações propostas

Este repositório será atualizado com frequência mínima semanal, com vistas à melhoria contínua da organização das pastas e arquivos, bem como à adição constante de novos conteúdos e exercícios.
